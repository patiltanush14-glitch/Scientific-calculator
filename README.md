# Scientific-calculator
I have created A GUI Scientific Calculator for students with tools like Python's Tkinter that provides a visual interface for performing both basic arithmetic and advanced mathematical operations.


---

## Features

### **Basic Operations**
- Addition (+)  
- Subtraction (−)  
- Multiplication (×)  
- Division (÷)


### **UI Features**
- Dark themed interface  
- Large display panel  
- Clickable buttons with modern colors  
- Error handling for invalid expressions  

---

## Tech Stack
- **Python 3.x**
- **Tkinter** (GUI module)
- **Math Library**
